Hexo
===
#Introduction
Hexo 是一個靜態網頁的產生系統，可將`markdown`編譯為`html`，由台灣人[tommy351](https://github.com/tommy351)所開發的
#Before

- `nvm` 
	> node版本控制
	
- `Node.js`
	> Hexo 是由node.js建構而成的
	
- `yarn` or `npm`
	> package 依賴控管
	
#Installaion
```sh 
	$npm install -g hexo-cli
	//全域安裝 Hexo
	$hexo init xxxx
	// xxx 是專案名稱
	$cd xxx
	//進入專案
	$npm install
	//安裝相關依賴
	$hexo server
	//啟動local  server
	//觀察是否安裝成功
```
